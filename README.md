# testeComissao

Teste Comissao é um programa simples feito com node, onde tem um metódo POST que vai receber o seguinte JSON
[
    {
        "vendedor": 1,
        "data": "2022-03-01",
        "mes": 3,
        "valor": "1740.18"
    },
    {
        "vendedor": 1,
        "data": "2022-04-01",
        "mes": 4,
        "valor": "5250.36"
    },
    {
        "vendedor": 2,
        "data": "2022-03-01",
        "mes": 3,
        "valor": "2100.36"
    },
    {
        "vendedor": 2,
        "data": "2022-04-01",
        "mes": 4,
        "valor": "3150.36"
    }
]
e irá calcular as devidas comissões de acordo com condições pré estipuladas.
